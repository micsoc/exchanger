Instukcja Api

  • Pobranie klucza Api

    • Ścieżka /api/* zabezpieczona jest kluczem JSON Web Token.

    Hasło do logowania to Jupikajej

    Aby pobrać klucz wyślij żądanie POST w Body przekazując dane logowania:

    {
    "login":"user"
    "password":"Jupikajej"
    }

    na adres: localhost:8080/logIn

    Wszystkie poniższe żądania wysyłaj z autentykacją "Bearer Token" oraz otrzymanym kluczem

    Klucz wygasa po 10 minutach

  • Pobranie listy dostępnych walut:

    Aby pobrac listę dostępnych walut wyślij żądanie GET na adres http://localhost:8080.com/api/list z pobranym wcześniej tokenem JWT

    • Curl:

    curl --location --request GET 'localhost:8080/api/list' \
    --header 'Content-Type: application/json' \
    --header 'Authorization: Bearer [JWT]' \
    --data-raw ''

  • Przeliczenie jednej waluty na inną

    Aby przeliczyć walutę wyślij metodą POST żądanie w formacie:

    {
    "amount":[ilość],
    "sell":[waluta sprzedawana],
    "buy":[waluta kupowana]
    }
    np. { "amount":"100", "sell":"GBP", "buy":"EUR" }

    na adres http://localhost:8080/api/exchange

    • Curl:

    curl --location --request POST 'localhost:8080/api/exchange' \
    --header 'Content-Type: application/json' \
    --header 'Authorization: Bearer [JWT]' \
    --data-raw '{

    "amount":"100",

    "sell":"EUR",

    "buy":"PLN"

    }'

  • Porbanie aktualnych kursów dla listy walut

    Aby pobrać aktualne kursy walut wyślij tablicę walut metodą POST na adres
    http://localhost:8080/api/courselist

    Curl:

    curl --location --request POST 'localhost:8080/api/courselist' \
    --header 'Content-Type: application/json' \
    --header 'Authorization: Bearer [JWT] --data-raw '["USD","PLN","CHF","GBP","CZK","DKK"]'

  • Logi

    Aby sprawdzićistniejące logi wyślij żądanie GET na adres localhost:8080/logs/all
