Instukcja Api

  • Pobranie listy dostępnych walut:

    Aby pobrac listę dostępnych walut wyślij żądanie GET na adres http://miscoc.herokuapp.com/api/list

  • Przeliczenie jednej waluty na inną

    Aby przeliczyć walutę wyślij metodą POST żądanie w formacie:

    {"amount":[ilość],
    "sell":[waluta sprzedawana]}
    "buy":[waluta kupowana],

    np. { "amount":"100", "sell":"GBP", "buy":"EUR" }

    na adres micsoc.herokuapp.com/api/exchange

  • Porbanie aktualnych kursów dla listy walut

    Aby pobrać aktualne kursy walut wyślij tablicę walut metodą POST na adres
    http://micsoc.herokuapp.com/api/courselist
